用github+hexo搭建个人博客

1搭建博客
1）注册github账号
2）下载git、nodejs
3）验证安装成功，win+R打开命令提示符（cmd），输入git –version、node -v（若git显示不是内部命令则添加环境变量即可）
4）在任意位置右键运行git bash here 输入npm install hexo -g 安装hexo 输入hexo -v检测是否安装成功
5）安装hexo依赖，npm install –save hexo-deployer-git
6）在GitHub中新建仓库名称格式为“用户名/用户名.github.io”,在GitHub中新建ssh密匙，标题随意，配置ssh密匙cd ~/.ssh查看是否有ssh密匙；生成sshkey，输入ssh-keygen -t rsa -C “邮箱地址”后，连续三次回车即可,在C盘用户文件夹下生成了.ssh文件点进去用记事本打开id_rsa.pub,将里面内容复制到GitHub建立的ssh密匙标题下内容中。
用ssh -T git@github.com 测试是否成功，输入yes
7）配置账号和密码
git config –global user.name “user.name”
git config –global user.email “user.email”
8）新建博客存放目录，里面必须为空，进入输入初始化命令hexo init,可看到目录下生成文件
9）hexo s 可在浏览器访问本地静态博客网页啦
10）编辑博客存放目录下_config.yml文件，添加
deploy：
type: git
repository: git@github.com:用户名/用户名.github.io.git
branch: main/master（现在都是main）
11)hexo d部署到公网，此时很容易出现链接不上的问题，是现在计算机安全性更高不信任任何私自站点链接，输入git config –global http.sslVerify false //解决连接不上的问题，由于不信任任何站点
即可进入github连接认证阶段输入密码进行ssh链接即可在浏览器用”用户名.github.io”访问站点
